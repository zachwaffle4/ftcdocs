TensorFlow Archive
-----------------

The :doc:`TensorFlow Archive <tensorflow/index>` contains resources related to TensorFlow Object Detection (TFOD) that were used in previous FTC seasons. This includes documentation for CENTERSTAGE (2023-2024), POWERPLAY (2022-2023), and FREIGHT FRENZY (2021-2022), as well as programming resources for using TensorFlow in Blocks and Java.
